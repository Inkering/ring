This is a webring, a set of websites with a common theme. In this case, the theme circles around homes. It exists to share ideas, traffic, questions, and to build community. If you have a: portfolio, diary, blog, or wiki about your life // what you care about, it would be cool if you joined the ring! to add yourself, make a pull request. Learn more about joining the webring on github. Or, contact someone in the ring currently for help making an internet home! Putting a crop of the header image linking to the ring to your site will help visitors navigate.
